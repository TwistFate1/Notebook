## C++ 简介
- C 语言代表的过程性语言
- C++ 在 C 语言基础上添加的类代表的面向对象语言
- C++ 模板支持的泛型编程

## C++ 简史
`面向对象编程（OOP）`

### C 语言
- **UNIX** 操作系统 `高级语言，致力于解决问题`
- 汇编语言 `低级语言，直接操作硬件`

### C 语言编程原理
- 数据 + 算法 = 程序 `过程化编程，注重算法`
- 结构化编程 `易于组织和理解`
- 自顶向下（top-down）设计 `大型程序分解成小型、便于管理的任务`

### 面向对象编程
- OOP 概念 `强调数据`

### C++ 和泛型编程
`独立于类型的代码`
