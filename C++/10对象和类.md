# 对象和类

## 抽象和类

### C++ 中的类

> 将实现细节放在一起并将它们与抽象分开被称为封装。

```cpp
class World {
private: // 可以隐藏
    float mass;
    char name[20];
public:
    void tellall(void);
    ...
};  
```

### 实现类成员函数

- 定义成员函数时，使用作用域解析运算符（::）来标识函数所属的类。
- **类方法可以访问**类的 private 组件。

> 在 OOP 中，调用成员函数被称为发送消息。
>
> 典型的类声明的格式：

```C++
class className {
private:
    data member declarations
public:
    member function prototypes
};
```
