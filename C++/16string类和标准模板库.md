# string 类和标准模板库

## string 类

> 具有丰富的功能。

## 智能指针模板类

> 离开代码块，指针自动过期。

- auto_ptr

    > 可能存在问题，两个指针指向一个对象。

- shared_ptr

    > 引用计数，最后一个指针过期时，才使用 delete。

- unique_ptr

    > 只能有一个智能指针可拥有它。

## 标准模板库

> STL 提供了一组表示容器、迭代器、函数对象和算法的模板（泛型编程）。

**注意：** 区间范围左闭右开。

## 泛型编程

### 迭代器类型

1. 输入迭代器

    > **只读**，单向迭代器，可以递增，但不能倒退。

2. 输出迭代器

    > **只写**。

3. 正向迭代器

    > **读写**或**只读**。

4. 双向迭代器

    > 顾名思义。

5. 随机访问

    > 能够调到容器中的任何一个元素。
