# 循环和关系表达式

## for 循环

### for 循环的组成部分

1. 设置初始值。
2. 执行测试，看看循环是否应当继续进行。
3. 执行循环操作。
4. 更新用于测试的值。

```C++
for (initialization; test-expression; update-expression)
    body
```

> C++ 将整个 for 看作一条语句。

### 递增运算符（++）和递减运算符（--）

> 前缀版本的效率比后缀版本高。
>
> 后缀版本的优先级比前缀版本高。

#### 组合赋值运算符

| 操作符 | 作用（L 为做操作数，R 为右操作数） |
| :-: | :-: |
| += | 将 L + R 赋给 L |
| -= | 将 L - R 赋给 L |
| *= | 将 L * R 赋给 L |
| /= | 将 L / R 赋给 L |
| %= | 将 L % R 赋给 L |

### 复合语句

> 用两个花括号（{}）来构造一条符合语句（代码块）。

### 逗号运算符

```C++
i = 20, j = 2* i; // i set to 20, then j set to 40
```

### 关系表达式

#### 关系运算符

| 操作符 | 含义 |
| :-: | :-: |
| < | 小于 |
| <= | 小于或等于 |
| == | 等于 |
| > | 大于 |
| >= | 大于或等于 |
| != | 不等于 |

> 关系运算符的优先级比算术运算符低。

## while 循环


## 基于范围的 for 循环（C++11）

```C++
double prices[5] = {4.9, 10.99, 6.87, 7.99, 8.49};
for (double x : prices)
    cout << x << std::endl;
```

```C++
for (double &x : prices)
    x= x *0.80;
```

```C++
for (int x : {3, 5, 2, 8, 6})
    cout << x << " ";
cout << '\n';
```
