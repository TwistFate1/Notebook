## 进入 C++

### main()函数
```C++
#include <iostream>
int main()
{
	using namespace std;
	cout << "Come up and C++ me some time.";
	cout << endl;
	cout << "You won't regret it!" << endl;
	return 0; // terminate main()
}
```

1. 作为接口的函数头
1. main() 不能使用其他名称

### C++ 源代码的格式化
1. 源代码中的标记和空白
1. C++ 源代码风格
- 每条语句占一行
- 每个函数都有一个开始花括号和一个结束花括号，这俩个花括号各占一行
- 函数中的语句都相对于花括号进行缩进
- 与函数名称相关的圆括号周围没有空白

## 函数

### 使用有返回值的函数
```C++
double x;
x = sqrt(6.25);
```
被调用的函数叫作被调用函数，包含函数调用的函数叫作调用函数。

函数原型：
```C++
double sqrt(double);
```

**注意：** C++程序应当为程序中使用的每个函数提供原型。

### 函数变体
多参数
```C++
double pow(double, double);
```

不含参数
```C++
int rand(void);
```
