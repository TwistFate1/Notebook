# 函数

## 函数和数组

```C++
#include <iostream>
const int ArSize = 8;
int sum_arr(int arr[], int n); // 仅在函数头或原型中 int *arr 和 int arr[] 的含义相同
int main() {
    using namespqce std;
    int cookies[ArSize] = { 1,2,4,8,16,32,64,128 };

    int sum = sum_arr(cookies, ArSize);
    cout << "Total cookies eaten: " << sum << "\n";
    return 0;
}

int sum_arr(int arr[], int n) {
    int total = 0;
    for (int i = 0; i < n; i++) {
        total = total + arr[i];
    }
    return total;
}
```
